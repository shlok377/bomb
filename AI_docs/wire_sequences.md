# WIRE SEQUENCES
Red: 1st=C, 2nd=B, 3rd=A, 4th=A/C, 5th=B, 6th=A/C, 7th=A/B/C, 8th=A/B, 9th=B.
Blue: 1st=B, 2nd=A/C, 3rd=B, 4th=A, 5th=B, 6th=B/C, 7th=C, 8th=A/C, 9th=A.
Black: 1st=A/B/C, 2nd=A/C, 3rd=B, 4th=A/C, 5th=B, 6th=B/C, 7th=A/B, 8th=C, 9th=C.
(A/B/C means cut if connected to any of those letters).
