# Stage 3: Phase 1 Modules Integration

## Objectives
- Develop the first set of basic modules.
- Implement Phase 1 bomb generation.

## Tasks
1. **Module 1: Simple Wires**
   - Logic: Randomized wires (3-6), cutting mechanics based on manual rules.
2. **Module 2: The Button**
   - Logic: Color/text combinations, press-and-hold mechanics synced with the global timer.
3. **Module 3: Keypads**
   - Logic: 4 unpronounceable symbols selected from specific columns in the manual.
4. **Phase 1 Bomb Generation:**
   - Randomly select 3 of the available modules to render on the front face of the bomb in `phase1.html`.
