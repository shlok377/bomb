# Stage 4: Phase 2 Modules Integration

## Objectives
- Develop the next set of more complex modules.
- Implement Phase 2 bomb generation.

## Tasks
1. **Module 4: Simon Says**
   - Logic: Flashing sequences dependent on the serial number (vowel presence) and strike count.
2. **Module 5: Who's on First**
   - Logic: Two-step visual/text association puzzle.
3. **Module 6: Memory**
   - Logic: Multi-stage puzzle requiring tracking previous inputs and displays.
4. **Module 7: Morse Code**
   - Logic: Flashing light decoding to a specific frequency.
5. **Phase 2 Bomb Generation:**
   - Create `phase2.html` and logic to randomly select 4 modules from the available 7 for each level.
